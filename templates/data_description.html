Predictions Compare Nations View All Data Data Description

The data that can be viewed in this application is based off predictions done on historical data from the previous 3 Olympic Games results.
Descriptions of the dataset's used as well as the scripts used have been included below.

Data Description

Datasets

  • results.csv: Original dataset containing results from various Olympics.
  • updated_filtered_results.csv: Filtered dataset with unnecessary columns and data older than 2012 removed.
  • olympic_rank_counts.csv: Further cleaned dataset from remove_data.py script.
  • olympic_predictions_2024.csv: Predicted winners for each event from predict_script.py script.
  • processed_olympic_predictions_2024.csv: Final cleaned dataset with only the position of the nation for each specific event.

Data Cleaning and Engineering Steps

Step-by-step explanation of the data cleaning and engineering process:

  1. Loaded the original dataset results.csv.
  2. Removed unnecessary columns and filtered out data older than 2012, resulting in updated_filtered_results.csv.
  3. Used remove_data.py script to clean the data further and create olympic_rank_counts.csv.
  4. Predicted the winners for each event using predict_script.py and saved the results in olympic_predictions_2024.csv.
  5. Finally, used clean_data.py to clean the dataset further, keeping only the position of the nation for each event, resulting in processed_olympic_predictions_2024.csv.
